# py2go

目标：寻找一些有意思的 Python 包，用 Go 重新实现一下。

## 目录结构及描述
```bash
    ├── README.md
    ├── common // 常量以及配置定义
    │   ├── const.go
    │   └── error.go
    ├── compare // 实现效果比较
    ├── handlers // 各种 Python 包的实现
    ├── test // 测试用例目录 
    ├── utils // 工具函数目录
    ├── go.mod // 依赖包
    ├── main.go // main 函数
```
